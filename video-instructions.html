N.R.R.C Briefing Video Creation Guide

Note: This document provides instructions for recording and preparing the briefing video that plays at the start of the N.R.R.C escape room companion application.

Video Requirements

  • Duration: 1-2 minutes
  • Resolution: 1280x720 (720p) or higher
  • Format: MP4 (H.264)
  • File size: Keep under 10MB if possible for fast loading
  • Audio: Clear spoken dialogue with minimal background noise

Script

The full script is available in the script.md file in the repository. The script is structured as a briefing from Dr. Eliza Chen, the Director of Operations at the N.R.R.C facility.

Recording Options

Option 1: Recorded Actor Performance

  1. Find someone to play Dr. Eliza Chen
  2. Set up a simple "command center" backdrop (dark background, minimal technical props)
  3. Record the performance using the script
  4. Add simple visual effects for "transmission breaking up" at the end

Option 2: AI-Generated or Animated

  1. Use an AI video generator tool with the script
  2. Consider tools like Synthesia, D-ID, or HeyGen for AI presenter
  3. Or create a motion graphics animation with voiceover

Option 3: Simple Slideshow with Voiceover

  1. Create slides that match the script sections
  2. Record a voiceover following the script
  3. Combine into a video presentation

Post-Production

  • Add "N.R.R.C" branding and warning graphics
  • Include subtle tech interfaces/HUD elements that appear during relevant sections
  • Add alarm sounds at the beginning
  • Apply a slight "digital distortion" filter for the tech feel
  • End with transmission cutting out effect

File Placement

Once your video is complete:

  1. Name the file briefing-video.mp4
  2. Place it in the assets/ directory
  3. Create a poster image (first frame or custom image) named video-poster.jpg in the same directory

Testing

Before finalizing:

  • Test the video in the application to ensure it loads properly
  • Verify that both the "video ended" and "skip video" functions work correctly
  • Check that the transition to the boot sequence is smooth
Important: Keep in mind that players will be viewing this at the start of their escape room experience, so it should set the tone and provide essential context without being too long or complex.
